# Конструктор резюме

Одностраничное веб-приложение для быстрого создания и выгрузки PDF-резюме без серверной части. Интерфейс редактируется прямо в браузере: можно менять текстовые блоки, добавлять или удалять записи об образовании и опыте, загружать своё фото и сохранять результат как файл формата A4.

## Как открыть

- **Локально:** достаточно открыть `index.html` в браузере.
- **GitHub Pages:** разместите файлы репозитория в ветке `main` (или `gh-pages`) и включите GitHub Pages в настройках. Главная точка входа — `index.html`, а стили и скрипты лежат в `assets/`.

## Структура

- `index.html` — разметка приложения и подключение зависимостей (Google Fonts, html2pdf.js).
- `assets/style.css` — стили интерфейса редактора и макета резюме.
- `assets/script.js` — логика связывания полей, добавления/удаления блоков и выгрузки в PDF.

## Использование

1. Введите данные в левой панели или редактируйте текст прямо в макете резюме.
2. Кликните по заглушке фото или по кнопке «Загрузить фото», чтобы добавить собственное изображение.
3. Нажмите «Скачать PDF», чтобы получить одностраничный файл резюме формата A4.
